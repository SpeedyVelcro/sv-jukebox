# SV Jukebox

Rudimentary Godot addon for playing music. The main attraction is a jukebox
UI for playing game soundtracks in-game from the menu.

![Screenshot of jukebox UI](readme_screenshot.png)

## Dependencies
- Requires [Godot 4.7](https://godotengine.org/download/4.x)

## Installation
- Download the archive `sv-jukebox-addon-vX.X.X.zip` from the
  [latest release page](https://github.com/SpeedyVelcro/sv-jukebox/releases/latest).
- Extract the archive into the root directory of your project (i.e. the
  same folder as your `project.godot` file).
- Enable the `SV Jukebox` plugin under `Project > Project Settings... > Plugins`
